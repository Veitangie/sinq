# Contributing to Sinq

Thank you for your interest in making sinq a better tool for everyone!

At this time I'm not accepting any pull requests or external code contributions for this project. However, if you still think you have a vision or an idea of how sinq could become better - you're always welcome to contribute in other ways:
- join discussions, 
- report bugs and vulnerabilities (make sure you read the [policy](SECURITY.md) first),
- open issues for feature requests.

This project is currently operated by me alone, so while I do believe in the open source community, I am not ready to take on the burden of managing external contributions alongside the overall project management. This is not a decision taken lightly, and I leave the door open for the future changes to contributions policy. 

And finally, as the project is licensed under GPLv3, you're always welcome to fork the repository and use the code for your own purposes, who's going to stop you?
